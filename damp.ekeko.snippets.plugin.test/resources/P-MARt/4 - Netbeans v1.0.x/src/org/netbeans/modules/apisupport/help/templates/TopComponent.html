A top-level window in the IDE. Can be docked, set on workspaces, and its position saved. This template includes hints for developing various areas of functionality:
  • Creating the component, possibly making it cloneable, giving it a name, icon, context help, etc.
  • Visually developing the component's content using the Form Editor.
  • Associating it with a particular mode and workspace.
  • Making it persistent, i.e. capable of surviving IDE restarts.
  • Making it interact well with IDE actions of all sorts.
